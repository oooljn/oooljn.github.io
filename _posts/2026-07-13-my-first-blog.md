## 一、实践目标

本次实践的目标是创建一个公开的 GitHub 仓库，将个人 Blog 内容保存到仓库中，并通过 GitHub Actions 自动发布到 GitHub Pages 网站。

## 二、Git 与 GitHub

Git 是一个分布式版本控制工具，可以记录文件的修改历史。

GitHub 是一个基于 Git 的代码托管平台，可以用于保存项目、管理版本和开展协作。

在本次实践中，我使用 GitHub 仓库保存博客的 Markdown 源文件，并使用 GitHub Pages 发布网站。

## 三、Markdown

Markdown 是一种轻量级标记语言，可以使用简单的符号完成文章排版。

例如：

- 使用 `#` 创建标题；
- 使用 `**文字**` 表示粗体；
- 使用 `-` 创建无序列表；
- 使用三个反引号创建代码块。
